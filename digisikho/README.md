# 📚 DigiSikho — आसान सीखो, आगे बढ़ो

A MERN stack website to help elderly Indian parents learn mobile apps through Hindi YouTube tutorials.

## 🚀 How to Run

### Requirements
- Node.js (v16 or higher)
- npm

### Steps

1. **Install server dependencies:**
```
cd server
npm install
```

2. **Install & build React frontend:**
```
cd ../client
npm install
npm run build
```

3. **Start the server:**
```
cd ../server
node index.js
```

4. **Open in browser:**
```
http://localhost:5000
```

That's it! The website runs on port 5000.

## 📱 Apps Covered
- WhatsApp, PhonePe, Paytm, Instagram, YouTube, Google Maps, Google Pay, Truecaller, DigiLocker

## ✨ Features
- Real Hindi YouTube tutorial videos embedded
- Original app icons from Wikipedia
- Step-by-step checklist with progress tracking
- Category filters & search
- Safety tips page
- Emergency helpline numbers
- Mobile responsive design
- Bilingual (Hindi + English)
