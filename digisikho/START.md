# DigiSikho - आसान सीखो, आगे बढ़ो

## How to Start the Website

### Step 1 - Install server packages
```
cd server
npm install
```

### Step 2 - Install and build React frontend
```
cd ../client
npm install
npm run build
```

### Step 3 - Start the server
```
cd ../server
node index.js
```

### Step 4 - Open in browser
```
http://localhost:5000
```
